一、        说明

 

（一）   绩效指标

绩效指标是衡量绩效目标实现程度的考核工具。绩效指标的确定应当遵循以下原则：

1）是具体，即指标定义简洁明确，不产生歧义；

2）是可衡量，即指标有确定的评价基准，可以对其进行衡量和分析；

3）是可实现，即指标在现行条件下可以收集到相关证据；

4）是相关，即指标与项目某方面的核心问题有关，能够为管理提供有用信息；

5）是时限，即指标有确定的时间界限。

绩效指标可分为产出指标、成效指标、效率指标三类。其中产出指标反映项目实施提供的产品或服务；成效指标反映项目产出或活动带来的结果；效率指标是指通过获取、使用和管理资源以实现项目结果或产出的经济性。

例如，对于某促进就业培训项目而言，参与培训人数是一个产出指标，反映项目提供服务的情况；参与人员就业率是一个成效指标，反映实施项目的结果；参与者人均成本是一个效率指标，反映服务提供的经济性。

（二）   绩效目标

绩效目标是用指标表示项目的计划绩效水平，包括用量化的标准、数值或比率表示的使用财政资金达到的产出和效果。绩效目标的确立应遵循以下原则：

1）是指向明确。即绩效目标要符合区域经济社会发展的总体规划、部门职能和事业发展规划，并与相应的财政支出范围、方向、效果紧密结合。

2）是具体细化。即绩效目标应当从数量、质量、成本和时效等方面进行细化，尽可能量化，如无法量化，可以采用定性的分级分档形式表述。

3）是合理可行。即绩效目标的制定要经过科学预测和调查研究，既要符合实际，又要有一定的挑战性（即高于历史水平或平均水平，在既定条件下经过努力可以实现）。

例如某个交通安全项目，它有以下绩效目标：

n  高速公路交通事故死亡率降低到亿分之一

n  商业航空运输重大事故率降低到百万分之0.1

n  铁路运输事故和意外中断率降低到16.14%

n  市内交通安全事故死亡率降低到亿分之0.488

n  管道运输总事故数降低到每年307次

（三）   绩效标准

绩效标准是衡量项目绩效的标尺和准绳，一般用一个数值范围表示。绩效标准一般有行业标准、计划标准、历史标准、经验标准四种。具体选用哪种标准，主要根据项目的特点来确定。

n  计划标准：是以事先制定的目标、计划、预算、定额等预计数据作为财政支出绩效的标准。（污水集中处理率）

n  行业标准：是以一定行业许多群体的相关指标数据为样本，运用数理统计方法计算和制定出的该行业评价标准。建立在基础数据较为完备的前提下。（空气质量）

n  历史标准：是以本地区、本部门、本单位或同类部门、单位、项目的绩效指标的历史数据作为样本，运用一定的统计学方法计算出各类指标的平均历史水平作为评价的标准。（政府部门人均能耗）

n  经验标准：是根据长期的财政经济活动发展规律的管理实践，由在专业领域有丰富经验的专家学者，在经过严密分析研究后得出的有关指标标准。（吨水处理电耗）

每一绩效目标的标准分为优、良、中、差四个等级。
